# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x.x   | ✅        |

This project is in active development. Only the latest release receives security updates.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them privately via GitHub's security advisory feature:
**[Report a vulnerability](https://github.com/kubeswarm/kubeswarm/security/advisories/new)**

You can expect an acknowledgement within **48 hours** and a status update within **7 days**.

If the vulnerability is accepted, we will publish a fix and a public advisory together.
If it is declined, we will explain why.
